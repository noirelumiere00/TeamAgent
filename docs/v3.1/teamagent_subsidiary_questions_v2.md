# 子会社エンジニア向け 質問リスト v2

**目的**：TeamAgent v3.1（OpenClaw フル採用版）を構築するにあたり、子会社の OpenClaw 120ユーザー運用実績から実務ノウハウを吸い上げる。
**形式**：テキストベースでの質問リスト送付（先方多忙のため対面 MTG は厳しい想定）
**回答粒度**：1問あたり数行〜数十行で OK。「実装の苦労」「失敗談」「気をつけてること」を中心に。
