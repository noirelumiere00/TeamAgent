2.何ができるか — 5機能を営業視点で1行に

Slack だけ開いていれば、営業は新しいツールを覚える必要なく以下の5つを使えます。色は実装フェーズの順に並んでいます。

🎬
機能 ④ ／ Phase 1
動画 URL 分析
TikTok / YouTube Shorts URL を投げると、いいね・保存・フック・訴求軸など 6観点で分析。
📬
機能 ② ／ Phase 2
Mail ワークフロー
毎朝、返信が必要なメールを整理 + 案 A/B/C を提案、ボタンで Gmail 下書きに保存。
📊
機能 ⑤ ／ Phase 2.5
営業進捗サマリー
マネージャーに毎週、各営業の 案件進捗・受注角度・要フォロー を自動配信。
📝
機能 ③ ／ Phase 3
提案コンテンツ生成
クライアント名・商材・URL を投げると、5フェーズで MD/HTML 提案資料を5割→8割と対話で生成。
💡
機能 ① ／ Phase 4
自動サジェスト
@TeamAgent で過去事例を呼び出すと、本人だけに見える形で関連事例 Top3 を3秒で提示。

③ フック　「実は 90% の人が知らない…」（数字提示型）
④ 訴求軸　左脳訴求（数値・論理で説得）

6.何を承認してほしいか — 次のアクション

上司への承認依頼事項（5 点）

  • ④ 営業 OAuth 接続：営業 16 名分の Gmail OAuth 接続承認（社内方針として全員合意済み前提）
